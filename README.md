# Personal Portfolio + Blog

A **small, manageable project** built with **Next.js** and **TypeScript** to showcase design skills, dynamic routing, and responsive web development.  

This project demonstrates my ability to build a modern portfolio with a blog section, using clean code and deployable architecture.

---

## ✅ Why This Project

- Small and focused — easy to understand and expand.  
- Demonstrates **design and UI skills**.  
- Uses **Next.js pages** and **dynamic routing** for blog posts.  
- Fully **TypeScript** typed for better maintainability.  
- Ready for **deployment** and responsive across devices.  

---

## 🛠️ Skills Demonstrated

- **Next.js Pages & Routing** – Dynamic blog routes and multiple pages.  
- **Responsive Design** – Works on mobile, tablet, and desktop.  
- **Deployment** – Configured for hosting on Vercel or other platforms.  
- **TypeScript** – Strongly typed components for safer code.
